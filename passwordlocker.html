    Behind History

    When I joined college, I had to create several accounts in various online tutorial platforms, coding platforms etc. Then a time came when I started to forget passwords even sometimes it happened that I totally forgot that I have an account in that platform. To work around this, I made a text file and stored it in a private cloud vault. But when it was needed, it was very time consuming to access a password from that text file. So I decided to make a web app to store and access passwords very easily. That time I was learning Django. So I started to make this Password Locker using Django.

    What can it do?

    It can
    • store Passwords
    • modify existing Passwords / User Name / Site name
    • add new Sites, Passwords
    • delete Passwords / User Name / Site name

    How I made it?

    Backend is implemented using Python/Django and HTML, CSS, Bootstrap, jquery and AJAX is used for frontend.
